# Exercise: Square Area

Write a program that calculates the area of a square.

## Task
- Read the side length of a square
- Calculate and display the area of the square

## Examples
**Example 1:**
```
Enter side length: 7
The area is: 49
```

**Example 2:**
```
Enter side length: 12
The area is: 144
```

**Example 3:**
```
Enter side length: 5
The area is: 25
```
