# Exercise: Rectangle Area

Write a program that calculates the area of a rectangle.

## Task
- Read the length and width of a rectangle (as `float`)
- Calculate and display the area with `.2f` formatting

## Examples
**Example 1:**
```
Enter length: 5.3
Enter width: 3.2
The area is: 16.96
```

**Example 2:**
```
Enter length: 10.5
Enter width: 7.8
The area is: 81.90
```

**Example 3:**
```
Enter length: 4.0
Enter width: 6.5
The area is: 26.00
```

## Formula
`Area = length Ã— width`
