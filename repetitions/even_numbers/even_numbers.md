# Even Numbers

Print the even numbers up to a given number.

## Task
- Read a positive integer `n`
- Print the even numbers from 2 up to `n` (inclusive)
- Each number on its own line
- If `n` is odd, stop at the last even number below it

## Examples
**Example 1:**
```
10
```
```
2
4
6
8
10
```

**Example 2:**
```
7
```
```
2
4
6
```

**Example 3:**
```
2
```
```
2
```
