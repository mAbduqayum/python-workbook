# Binary to Decimal

Convert a binary (base 2) number to decimal (base 10).

## Task
- Read a binary number from the user as a string
- Convert it to decimal by processing each digit
- Display the equivalent decimal number

## Examples
**Example 1:**
```
1010
```
```
10
```

**Example 2:**
```
1111
```
```
15
```

**Example 3:**
```
10000
```
```
16
```
