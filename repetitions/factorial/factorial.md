# Factorial

Compute the factorial of a given number.

## Task
- Read an integer `n` (`n >= 0`)
- Using a loop, compute `n!` (the product `1 * 2 * ... * n`)
  - Note: `0! = 1`
- Print the result

## Examples
**Example 1:**
```
0
```
```
1
```

**Example 2:**
```
5
```
```
120
```

**Example 3:**
```
10
```
```
3628800
```
