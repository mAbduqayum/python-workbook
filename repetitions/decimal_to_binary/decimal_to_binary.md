# Decimal to Binary

Convert a decimal (base 10) number to binary (base 2).

## Task
- Read a decimal number from the user as an integer
- Convert it to its binary representation
- Display the binary representation

## Examples
**Example 1:**
```
10
```
```
1010
```

**Example 2:**
```
15
```
```
1111
```

**Example 3:**
```
42
```
```
101010
```
