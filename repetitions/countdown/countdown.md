# Countdown

Print a countdown from a given number down to 1.

## Task
- Read a positive integer `n`
- Print the numbers from `n` down to 1
- Each number on its own line

## Examples
**Example 1:**
```
1
```
```
1
```

**Example 2:**
```
3
```
```
3
2
1
```

**Example 3:**
```
5
```
```
5
4
3
2
1
```
