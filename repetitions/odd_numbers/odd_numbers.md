# Odd Numbers

Print the odd numbers up to a given number.

## Task
- Read a positive integer `n`
- Print the odd numbers from 1 up to `n` (inclusive)
- Each number on its own line

## Examples
**Example 1:**
```
9
```
```
1
3
5
7
9
```

**Example 2:**
```
6
```
```
1
3
5
```

**Example 3:**
```
1
```
```
1
```
