# Sum to N

Compute the sum of the first `n` positive integers.

## Task
- Read a positive integer `n`
- Using a loop, compute the sum `1 + 2 + ... + n`
  - Do not use the formula `n * (n + 1) / 2`
  - Do not use the built-in `sum()` function
- Print the total

## Examples
**Example 1:**
```
1
```
```
1
```

**Example 2:**
```
5
```
```
15
```

**Example 3:**
```
100
```
```
5050
```
