# First N Numbers

Print the numbers from 1 up to a given number.

## Task
- Read a positive integer `n`
- Print the numbers 1 through `n`
- Each number on its own line

## Examples
**Example 1:**
```
1
```
```
1
```

**Example 2:**
```
3
```
```
1
2
3
```

**Example 3:**
```
5
```
```
1
2
3
4
5
```
