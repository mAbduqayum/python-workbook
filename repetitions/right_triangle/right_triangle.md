# Right Triangle Art

Create a right triangle pattern using asterisks.

## Task
- Read a positive integer `n` from the user
- Display a right triangle with `n` rows
- Each row `i` should contain `i` asterisks

## Examples
**Example 1:**
```
5
```
```
*
**
***
****
*****
```

**Example 2:**
```
3
```
```
*
**
***
```
