# Exercise: Rectangle Area

Write a program that calculates the area of a rectangle given its length and width in meters.

## Examples

**Example 1:**

```
5.3
3.2
```

```
16.96
```

**Example 2:**

```
10.5
7.8
```

```
81.90
```

**Example 3:**

```
4.0
6.5
```

```
26.00
```

## Formula

`Area = length × width`
