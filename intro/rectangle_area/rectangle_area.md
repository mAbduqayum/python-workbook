# Exercise: Rectangle Area

Write a program that calculates the area of a rectangle.

## Task

- Read the length and width of a rectangle (as `float`)
- Calculate and display the area with `.2f` formatting

## Examples

**Example 1:**

```
5.3
3.2
```

```
The area is: 16.96
```

**Example 2:**

```
10.5
7.8
```

```
The area is: 81.90
```

**Example 3:**

```
4.0
6.5
```

```
The area is: 26.00
```

## Formula

`Area = length × width`
