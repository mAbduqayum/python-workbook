# Exercise: Area of a Triangle

Write a program that calculates the area of a triangle using base and height.

## Examples

**Example 1:**

```
10.5
5.2
```

```
Area of triangle: 27.30
```

**Example 2:**

```
8.3
12.1
```

```
Area of triangle: 50.22
```

**Example 3:**

```
15.0
7.8
```

```
Area of triangle: 58.50
```

## Formula

`Area = (base × height) / 2`
