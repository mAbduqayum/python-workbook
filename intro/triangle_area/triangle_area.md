# Exercise: Area of a Triangle

Write a program that calculates the area of a triangle given its base and height in meters.

## Examples

**Example 1:**

```
10.5
5.2
```

```
27.30
```

**Example 2:**

```
8.3
12.1
```

```
50.22
```

**Example 3:**

```
15.0
7.8
```

```
58.50
```

## Formula

`Area = (base × height) / 2`
