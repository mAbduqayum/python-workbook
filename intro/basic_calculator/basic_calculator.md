# Exercise: Basic Calculator

Write a program that performs basic arithmetic operations on two numbers.

## Task

- Read two numbers (as `int`)
- Calculate and display addition and subtraction

## Examples

**Example 1:**

```
12
8
```

```
12 + 8 = 20
12 - 8 = 4
```

**Example 2:**

```
25
15
```

```
25 + 15 = 40
25 - 15 = 10
```

**Example 3:**

```
7
3
```

```
7 + 3 = 10
7 - 3 = 4
```
