# Exercise: Basic Calculator

Write a program that reads two integers and displays their sum and difference.

## Examples

**Example 1:**

```
12
8
```

```
12 + 8 = 20
12 - 8 = 4
```

**Example 2:**

```
25
15
```

```
25 + 15 = 40
25 - 15 = 10
```

**Example 3:**

```
7
3
```

```
7 + 3 = 10
7 - 3 = 4
```
