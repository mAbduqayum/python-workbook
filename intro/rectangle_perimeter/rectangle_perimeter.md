# Exercise: Rectangle Perimeter

Write a program that calculates the perimeter of a rectangle given its length and width in meters.

## Examples

**Example 1:**

```
8.5
5.2
```

```
27.40
```

**Example 2:**

```
12.3
4.7
```

```
34.00
```

**Example 3:**

```
6.0
6.0
```

```
24.00
```

## Formula

`Perimeter = 2 × (length + width)`
