# Exercise: Rectangle Perimeter

Write a program that calculates the perimeter of a rectangle.

## Examples

**Example 1:**

```
8.5
5.2
```

```
The perimeter is: 27.40
```

**Example 2:**

```
12.3
4.7
```

```
The perimeter is: 34.00
```

**Example 3:**

```
6.0
6.0
```

```
The perimeter is: 24.00
```

## Formula

`Perimeter = 2 × (length + width)`
