# Exercise: Sphere Volume

Write a program that calculates the volume of a sphere.

## Task

- Read the radius of a sphere (as `float`)
- Calculate and display the volume

## Examples

**Example 1:**

```
3
```

```
The volume is: 113.10
```

**Example 2:**

```
5
```

```
The volume is: 523.60
```

**Example 3:**

```
2
```

```
The volume is: 33.51
```

## Formula

`Volume = (4/3) × π × radius³`

## Note

Use `.2f` formatting to display 2 decimal places. Use `math.pi` for π
