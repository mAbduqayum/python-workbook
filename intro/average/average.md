# Exercise: Average Calculator

Write a program that calculates the average of three numbers.

## Examples

**Example 1:**

```
10
20
30
```

```
The average is: 20.00
```

**Example 2:**

```
5
15
25
```

```
The average is: 15.00
```

**Example 3:**

```
8
12
16
```

```
The average is: 12.00
```
