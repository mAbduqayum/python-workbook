# Exercise: Sum of the First n Positive Integers

Write a program that calculates the sum of all positive integers from 1 to n using the mathematical formula.

## Examples

**Example 1:**

```
5
```

```
15
```

**Example 2:**

```
10
```

```
55
```

**Example 3:**

```
100
```

```
5050
```

## Formula

`sum = n × (n + 1) / 2`
