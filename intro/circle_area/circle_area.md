# Exercise: Circle Area

Write a program that calculates the area of a circle.

## Task

- Read the radius of a circle (as `float`)
- Calculate and display the area

## Examples

**Example 1:**

```
5.2
```

```
The area is: 84.95
```

**Example 2:**

```
3.7
```

```
The area is: 43.01
```

**Example 3:**

```
10.0
```

```
The area is: 314.16
```

## Formula

`Area = π × radius²`

## Note

Use `.2f` formatting to display 2 decimal places. Use `math.pi` for π
