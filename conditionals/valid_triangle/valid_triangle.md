# Valid Triangle

Write a program that determines if three positive side lengths can form a valid triangle, per the inequality below.

## Triangle Inequality Theorem

For sides `a`, `b`, and `c` to form a valid triangle, all three must hold:

- `a + b > c`
- `a + c > b`
- `b + c > a`

## Examples

**Example 1:**

```
3
4
5
```

```
valid triangle
```

**Example 2:**

```
1
2
5
```

```
invalid triangle
```

**Example 3:**

```
1
1
3
```

```
invalid triangle
```
