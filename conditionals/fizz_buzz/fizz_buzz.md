# Fizz Buzz

Write a program that implements the FizzBuzz logic for a single number.

## FizzBuzz Rules
- If divisible by `3` only: `Fizz`
- If divisible by `5` only: `Buzz`
- If divisible by both `3` and `5`: `FizzBuzz`
- Otherwise: display the number itself

## Examples
**Example 1:**
```
15
```
```
FizzBuzz
```

**Example 2:**
```
9
```
```
Fizz
```

**Example 3:**
