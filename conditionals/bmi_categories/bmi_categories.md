# BMI Categories

Write a program that calculates `BMI` and determines the weight category.

## BMI Categories
| BMI Range           | Category      |
|---------------------|---------------|
| `BMI < 18.5`        | Underweight   |
| `18.5 ≤ BMI < 25.0` | Normal weight |
| `25.0 ≤ BMI < 30.0` | Overweight    |
| `BMI ≥ 30.0`        | Obese         |

## Examples
**Example 1:**
```
70
1.75

```

## Formula
`BMI = weight (kg) / height² (m²)`
