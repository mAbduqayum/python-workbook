# Between Values

Write a program that determines if a value `b` is between two other values `a` and `c`.

## Task
- Read three numbers `a`, `b`, and `c` from the user
- Determine if `b` lies between `a` and `c` (inclusive, regardless of whether `a < c` or `a > c`)
- Display `between` or `not between`

## Examples
**Example 1:**
```
5
7
10
```
```
between
```

**Example 2:**
```
10
7
5
```
```
between
```

**Example 3:**
```
5
12
10
```
```
not between
```

**Example 4:**
```
5
5
10
```
```
between
```

**Example 5:**
```
10
5
5
```
```
between
```
