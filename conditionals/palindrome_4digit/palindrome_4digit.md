# 4-Digit Palindrome

Write a program that determines if a `4-digit` integer is a palindrome.

## Examples
**Example 1:**
```
1221
```
```
palindrome
```

**Example 2:**
```
1234
```
```
not palindrome
```

**Example 3:**
```
9009
```
```
palindrome
```

**Example 4:**
```
7337
```
```
palindrome
```

**Example 5:**
```
5678
```
```
not palindrome
```
