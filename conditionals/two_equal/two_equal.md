# Two Equal

Write a program that determines if at least two of three integers are equal.

## Examples
**Example 1:**
```
5
5
3
```
```
yes
```

**Example 2:**
```
1
2
3
```
```
no
```

**Example 3:**
```
7
3
7
```
```
yes
```

**Example 4:**
```
4
4
4
```
```
yes
```

**Example 5:**
```
9
2
9
```
```
yes
```
