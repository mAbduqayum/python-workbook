# ignore/

Working files that live on the `solutions` branch only.

Anything in this directory is **stripped from `main`** by the
`Sync to Main` workflow (`.github/workflows/sync-main.yml`). Use it for
drafts, planning notes, and internal guides that students should never see —
e.g. `simplification-guide.md`.

Tests must not import from here; nothing in `ignore/` exists on `main`.
