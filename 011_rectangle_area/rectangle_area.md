# Exercise: Rectangle Area

Write a program that calculates the area of a rectangle.

## Task
- Read the length and width of a rectangle
- Calculate and display the area

## Examples
**Example 1:**
```
Enter length: 5
Enter width: 3
The area is: 15
```

**Example 2:**
```
Enter length: 10
Enter width: 7
The area is: 70
```

**Example 3:**
```
Enter length: 4
Enter width: 6
The area is: 24
```

## Formula
`Area = length Ã— width`
