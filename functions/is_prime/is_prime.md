# Is Prime

Write a function that determines if a number is prime.

## Task
- Create a function `is_prime(n)` that takes an integer
- Return `True` if the number is prime, `False` otherwise

## Examples

| Call           | Returns |
|----------------|---------|
| `is_prime(2)`  | `True`  |
| `is_prime(17)` | `True`  |
| `is_prime(1)`  | `False` |
| `is_prime(4)`  | `False` |
| `is_prime(29)` | `True`  |
| `is_prime(0)`  | `False` |
| `is_prime(-5)` | `False` |
