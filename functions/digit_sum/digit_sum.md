# Digit Sum

Write a function that calculates the sum of digits in an integer.

## Task
- Create a function `digit_sum(n)` that takes an integer
- Return the sum of all digits
- Don't convert the number to a string

## Examples

| Call              | Returns |
|-------------------|---------|
| `digit_sum(123)`  | `6`     |
| `digit_sum(0)`    | `0`     |
| `digit_sum(-456)` | `15`    |
| `digit_sum(1000)` | `1`     |
| `digit_sum(99)`   | `18`    |
