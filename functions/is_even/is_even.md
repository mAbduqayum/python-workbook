# Is Even

Write a function that determines if a number is even.

## Task
- Create a function `is_even(n)` that takes an integer
- Return `True` if the number is even, `False` otherwise

## Examples

| Call          | Returns |
|---------------|---------|
| `is_even(4)`  | `True`  |
| `is_even(7)`  | `False` |
| `is_even(0)`  | `True`  |
| `is_even(-2)` | `True`  |
| `is_even(-5)` | `False` |
